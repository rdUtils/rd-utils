String #

trim #

作用：

🎉 去除字符串空格

参数1为操作字符串str，参数2为类型type(1-前后空格 2-前空格 3-后空格 4-所有空格 默认为1）

Demo

typescript
import { trim } from '@rd/utils'

const str1 = trim(' a b c ', 1) // 'a b c'
const str2 = trim(' a b c ', 2) // 'a b c '
const str3 = trim(' a b c ', 3) // ' a b c'
const str4 = trim(' a b c ', 4) // 'abc'

removeSpecialChars #

作用：

🎉 移除给定字符串中的特殊字符,只保留字母、数字、空格和下划线

Demo

typescript
import { removeSpecialChars } from '@rd/utils'
const str = removeSpecialChars('Hello#_ are you? 123!$') // 'Hello_ are you 123'

pascalCase #

作用：

🎉 将字符串转为帕斯卡命名法（Pascal）, 即大驼峰式命名法

Demo

typescript
import { pascalCase } from '@rd/utils'

const str1 = pascalCase('abcd')  // Abcd
const str2 = pascalCase('ab_cd') // AbCd
const str3 = pascalCase('ab-cd') // AbCd
const str4 = pascalCase('ab#cd') // AbCd
const str5 = pascalCase('ab cd')) // AbCd
const str6 = pascalCase('ab2cd') // Ab2Cd
const str7 = pascalCase('_ab_cd_') // AbCd
